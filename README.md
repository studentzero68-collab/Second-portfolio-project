## What is inside

- Hero section — full bleed background, centered name and role, call to action buttons
- About Me card — two column layout with profile image, contact details, skill badges and bio
- Projects grid — 6 project cards with images, descriptions, tech tags and live links
- Contact section — social icons, contact form with HTML validation and contact details
- Footer — clean dark footer with my name and credit
- Bonus — auto dark mode using prefers-color-scheme and CSS variables

## Sections implemented

- Hero with sticky navbar and active underline state
- About Me with Bootstrap badge-style skill tags
- Projects grid — 3 columns on desktop, 2 on tablet, 1 on mobile
- Contact with form validation, social icons and contact details
- Dark footer
- Bonus: auto dark mode via prefers-color-scheme

## Rules followed

- Bootstrap 5 CSS only — no Bootstrap JS
- No JavaScript at all
- Semantic HTML landmarks used throughout — header, nav, main, section, footer
- All images have proper alt text
- HTML form validation only using required and type="email"
